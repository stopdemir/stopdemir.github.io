# stopdemir.github.io

Source for [stopdemir.com](https://www.stopdemir.com) — built with Jekyll (`jekyll-theme-minimal`) and hosted on GitHub Pages.

- `index.md` — About page (site root, `/`)
- `portfolio.md` — Portfolio page (`/portfolio/`)
- `about.md` — redirect stub, keeps the old `/about/` URL working
- `_layouts/default.html` — shared layout with the About/Portfolio tab nav
- `assets/css/style.scss` — site styling, including the sliding tab indicator
